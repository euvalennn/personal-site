# my personal website!
this repo contains all the code for my personal site!! it was built using [astro](https://astro.build), with no other additional UI frameworks added  
made by me (valen) with love ❤️

## features / pages
- **about me:** has a bit of info about me and the stuff i like doing as well as a few contact methods
- **travel map:** map of all places i've traveled to with short descriptions for each (you can access those by clicking on the name inside the marker popup)
- **guestbook:** people who visit my website can leave messages here!! it's hooked up to a [supabase](https://supabase.com) database
- **fun stuff:** a page with random stats and stuff i wanted to implement for fun lol it displays the song i'm currently listening (or last listened) to, the current weather on my city, a random cat image everytime you reload the page and it has a little space for doodling
